# Generador de Contraseñas en Python

Este proyecto genera contraseñas seguras de forma aleatoria usando Python.

## Características

- Permite elegir la longitud de la contraseña
- Genera caracteres aleatorios
- Utiliza letras, números y símbolos
- Incluye un indicador básico de seguridad

## Cómo ejecutar el programa

1. Descargar el archivo
2. Abrir la terminal
3. Ejecutar:

python password_generator.py
